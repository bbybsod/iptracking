# how to use 
### FOR TERMUX
```terminal
$ pkg update
$ pkg upgrade
$ pkg install git
$ pkg install python3
$ pip install requests
$ pip install pyfiglet
$ git clone https://github.com/bbybsod/iptracking.git
$ ls
$ cd iptracking
$ python3 iptrack.py
```

### FOR Linux ( Debian, Ubuntu, etc )
```terminal
$ sudo apt update
$ sudo apt upgrade
$ pkg install git
$ sudo apt install python3
$ pip install requests
$ pip install pyfiglet
$ git clone https://github.com/bbybsod/iptracking.git
$ ls
$ cd iptracking
$ python3 iptrack.py
```
